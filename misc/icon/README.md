# Icon

## 問題文

Hmmm... I can't find Flag easily. <br>
Hmm? There seems to be something written on the icon.


## 解法
PowerPointの設定ファイルからFlagを入手する問題。<br>

1. ファイルの拡張子を.zipに変更する。

2. docPropsフォルダの中にthumbnail.jpegというファイルがあり、これを開くとFlagが得られる。
