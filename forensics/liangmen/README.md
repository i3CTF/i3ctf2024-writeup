# Liangmen

## 問題文

There are days when you can't win at mahjong no matter how you play.



## 解法
拡張子に関する問題。<br>

1. ZIPファイルをunzipすると画像が得られる。

2. 画像内には「ZIP形式ではない」というヒントがある。
![](./img/step2.png)
3. ZIPファイルのバイナリを見ると、「PNG」と表記してあるので、拡張子を.pngにするとFlagが得られる。
![](./img/step3.png)
