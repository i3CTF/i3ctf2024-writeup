# pyc

## 問題文

0.1 + 0.2 = 0.3?



## 解法
pycファイルを逆コンパイルする問題。<br>

1. uncompyle6を使い逆コンパイルを行うとFlagを生成しているコードを得られる。


ちなみに、浮動小数点の関係で0.1 + 0.2 = 0.3は成立しない。

2進数では0.5,0.25など $\frac{1}{2^n}$ の小数の組み合わせしか表せないため、0.1などは近似値での計算となり結果が合わなくなってしまう。
